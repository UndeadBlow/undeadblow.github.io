OpenTranslate

Javascript, HTML, CSS

Расширение с открытым исходным для браузера Google Chrome, служащее для перевода слов и предложений с английского языка на русский и сохранения выбранных переводов в файл формата Anki. Anki является популярной программой для запоминания информации, использующей эффективный алгоритм и обладающей открытым исходным кодом. Кроме этого, расширение умеет искать транскрипцию слов и их аудио-файлы с произношением в Викимедиа.

ЮлмартМониторинг

C++ Qt

Программа для мониторинга цен в крупном онлайн-магазине Юлмарт. Программа способна искать товары по артикулу, добавлять их в базу данных и раз в заданный промежуток времени проверять изменения в количестве товара на складе, его цене и так далее. Поскольку у Юлмарта нет API, программа просто парсит исходный код страниц.
